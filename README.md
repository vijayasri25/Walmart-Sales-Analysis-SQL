## 📌 Project Overview

This project analyzes Walmart sales data using SQL and Tableau to identify sales trends, evaluate store performance, and understand the impact of holidays on weekly sales. The analysis is presented through an interactive Tableau dashboard.

---

## 🎯 Business Objectives

- Identify the top-performing stores.
- Analyze monthly and yearly sales trends.
- Compare holiday and non-holiday sales.
- Build an interactive dashboard for business users.

---

## 🛠️ Tools & Technologies

- MySQL Workbench
- SQL
- Tableau
- GitHub

---

## 📈 Dashboard Features

- Top 10 Performing Stores
- Monthly Sales Trend
- Yearly Sales Trend
- Holiday vs Non-Holiday Sales
- Interactive Filter Action
- Professional Dashboard Layout

---

## ❓ Business Requirements

- What is the total sales revenue?
- What is the average weekly sales?
- Which stores generated the highest sales?
- How do sales change over time?
- What is the impact of holidays on sales?

---

## 💡 Key Insights

- Store 20 generated the highest total sales.
- Non-holiday sales contributed significantly more than holiday sales.
- Sales showed seasonal trends across different months.
- Interactive filters allow users to analyze each store individually.
