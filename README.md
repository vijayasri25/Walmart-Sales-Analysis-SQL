# Walmart Sales Analysis Using SQL

## Project Overview

This project analyzes Walmart sales data using MySQL to identify sales trends, store performance, and the impact of holidays on sales.

## Dataset

Walmart Sales Dataset

## SQL Concepts Used

* Aggregate Functions
* GROUP BY
* ORDER BY
* LIMIT
* Date Functions
* Business Analysis Queries

## Business Questions Solved

1. Total Sales
2. Average Sales
3. Top 5 Stores
4. Bottom 5 Stores
5. Monthly Sales Trend
6. Yearly Sales Trend
7. Holiday vs Non-Holiday Sales
8. Store-wise Holiday Impact

## Key Insights

* Identified top-performing stores.
* Compared holiday and non-holiday sales.
* Analyzed monthly and yearly sales trends.
* Evaluated store-level sales performance.
